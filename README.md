# Bookshelf-API

## Cloning Project

```
git clone https://github.com/IrvanAhmadP/Bookshelf-API.git
cd Bookshelf-API
```

## Install Dependencies

Menggunakan pnpm
```
pnpm i
```

Menggunakan yarn
```
yarn
```

Menggunakan npm
```
npm install
```

## Menjalankan project

Menggunakan pnpm
```
pnpm start
```

Menggunakan yarn
```
yarn start
```

Menggunakan npm
```
npm run start
```
